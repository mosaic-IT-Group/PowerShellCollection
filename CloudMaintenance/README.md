# Cloud Maintenance Scripts

PowerShell scripts for cloud infrastructure maintenance and automation.

## Platforms

| Platform | Description |
|----------|-------------|
| [Azure](Azure/) | Microsoft Azure cloud maintenance scripts |

## Getting Started

Each cloud platform folder contains categorized scripts for common maintenance tasks:

- Resource management and cleanup
- Cost monitoring and optimization
- Security and compliance auditing
- Backup and recovery
- Monitoring and alerting
- Identity and access management
- Networking

## Requirements

- PowerShell 5.1+ or PowerShell 7+
- Platform-specific modules (see individual README files)
- Appropriate cloud permissions
